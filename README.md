# AI Chatbot for College Help Desk

## Project Overview
The AI Chatbot for College Help Desk is a system designed to help students get quick answers to common college-related questions. The chatbot uses Artificial Intelligence and Natural Language Processing (NLP) to understand user queries and provide accurate responses.

## Features
- Answer student queries instantly
- Provide information about courses, exams, and timetable
- Simple and user-friendly interface
- 24/7 availability for students
- Reduces workload for college staff

## Technologies Used
- Python
- Natural Language Processing (NLP)
- NLTK / TensorFlow
- Flask (for web application)
- HTML, CSS, JavaScript

## Project Structure
- chatbot.py – Main chatbot logic
- app.py – Flask web application
- templates/ – HTML files for interface
- static/ – CSS and JavaScript files

## How to Run the Project
1. Install Python
2. Install required libraries:
   pip install nltk flask
3. Run the application:
   python app.py
4. Open the browser and access the chatbot.
